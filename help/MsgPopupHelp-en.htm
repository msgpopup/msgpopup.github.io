What is MessagePopup II (MsgPopup II)
MessagePopup is Message Communication program in LAN for free of charge.
Simply Install & Send a message. Also can send a file and save logs .
It has Diary, Alarm . Everything is for making your work easy.

It is suitable for small office. not for big company. do not need server.

Requirements
  OS: windows 95/98/Me/2000/XP/Vista/7/8/10 and over ( XP and over is recommended) 
 LAN card : TCP/IP protocol installed .
 If use dynamic IP, make static IP with your network card MAC in DHCP server.
 Display resolution : At least 800 X 600, (over 1024 X 768 recommended).

On Windows XP or over version, should  allow access private network in the Firewall
 when you first install a MessagePopup, the window that you can allow access private networks pops up.
(XP) Click Unblock / (Win10, etc) check box of private network and click Allow access.
 If do not allow,  can not see users list.

How to use
Message Popup searches for Message Popup users on your network when starting,
or when you click the connection button.
When receive a message, it will popup with a sound.
To exit program, right click the system tray icon , and select exit.  
To make the popup window pop up , simply press[CTL+SHIFT]
To make the popup window disappear, simply press [Esc]   when activated. Anytime [CTL+SHIFT]  
Or click system close button
If you don't want window to popup when a message is received, right click the system tray icon,
and select  [Icon Window]
To see this manual at any time, press [F1]  or click   and help

Network Setting

When there are several different networks from my network IP.
( Ahead Three unit of IP is my network. X.X.X )

example>
Your computer A = 10.10.10.2
Same network   B = 10.10.10.3
Different network C,D = 10.10.20.3 , 10.10.20.4
Different networks S = 10.10.30.1 ~10.10.30.254

1)ping test to computer in different network
   On Dos command
   C:> ping 10.10.20.3
   C:> ping 10.10.30.1
     reply should be arrived.

2) Different network 10.10.20.X and 10.10.30.X
   (1)  Setup -> network tab , input 10.10.30 to different network .
   (2) And you do not want to communicate not all 10.10.20. network users.
        Just only two 10.10.20.3 and 10.10.20.4 are my concern.
       You do not want to let 10.10.20 network users know my name.
         -> Input two 10.10.20.3 , 10.10.20.4  ,and click [+].
   (3)  click to refresh network users.

  * Of course you can also input 10.10.30 to different network setting.
    But, if do that, searching time , it takes long time.
    So, If you want to communicate with only a few people in different network,
    IP function is useful.
   
   Then, What should do In computer  C and D?
     input 10.10.10 and 10.10.30 to different network setting.
    In S, 10.10.10.X -> different network setting
          10.10.20.3 and 10.10.20.4 -> IP Setting.

You can setup your Network with above functions.

Since version from Window XP, use the Windows Firewall.
The firewall block program uses the network.
So, to use the MessagePopup, you must allow access the personal network in Firewall.
By default, when you first install a MessagePopup, the window that you can allow access private networks pops up.
(XP) Click Unblock / (Win10, etc) check box of private network and click Allow access.

If you did not allow access permission at first executing, users list is not listed.
Solution) Windows Control Panel (Start-Control Panel, In case win10, right click Start icon- Control Panel)
-Security Center (System and Security) - Firewall
XP -Firewall  : Exceptions tab - check on box of MsgPopup, If do not have MsgPopup in the list, click Add Program, add MsgPopup
Win10, Etc- firewall : allow an app through firewall- click Change settings - check on boxes of MsgPopup name, private  public
(if MsgPopup not in the list, click Allow another app - Add MsgPopup)

  Language Support
  ver 4.0.0 support  language English, Korean
  But, If  you want to use other your language, you can translate it and send it by email to me ( msgpopup@empal.com)
  Then I will consider it to  apply to  update version.
  When you install ver 4.0 , there is a lang folder in the messagepopup main folder ( default C:\Program Files\MessagePopup\lang)
  english.txt  copy to your language.txt  (ex)  MessagePopup\lang\spanish.txt
  [language].txt format
   line 1  : [0] [1] [2] ... this words are reserved . cannot be changed . [0] : internal message  [1] : main window [2] Alarm window .... 
   line  2 :  English  .  change to Spanish   
   line number is important (ex) line #4  I am absent now.  change to  spanish.  Do not leave blank or skip line.
 (So I recommend to use Excel . just copy all and paste  it and  translate and copy all column and paste it to spanish.txt
   After translating, save it with encoding ANSI in notepad. Ignore warning.  Other encoding file cannot be read
   After translate it,  close msgpopup and re-execute messagepopup  and  setup  - language tab- Language : select spanish
  check language is changed correctly.

   This Translate is only for MessagePopupII , not for install  process language. Install language is English for default.
